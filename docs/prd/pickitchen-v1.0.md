# PicKitchen V1.0 产品需求文档

## 1. 项目概述

### 1.1 背景

基于市场C端应用发展趋势分析应用研究以及AI赛道快速发展现状，决定以C端AI生图业务为起点，同步开启IAP业务探索及AI中台能力积累，为后续快速迭代项目打下基础。

图片的模型基座能力仍期待完善，但AI生图对于传统图片处理给予技术侧更多的想象空间，可以低成本高效率提供围绕图片的丰富玩法，打造有趣的体验。

### 1.2 目标

- 完成AI中台搭建 V1.0（考虑可拓展性、通用性、实用性）
- 定义C端 AI-Human Photo Studio 产品 V1.0 版本

### 1.3 项目收益

- 验证C端IAP类型产品策略
- AI赛道生图方向建立产品站位

## 4. 功能详细说明

### 4.1 首页（P0）

#### 顶部Tab设计

| 配置项 | 说明 |
|--------|------|
| Tab类型 | 图片、视频（视频V1.0暂不展示） |
| 扩展性 | 支持多Tab框架，后续迭代Voice、Music |
| 服务端配置 | Tab顺序、Tab名称、开关状态 |
| 交互 | 支持滑动切换 |

#### Banner配置

| 配置项 | 说明 |
|--------|------|
| 跳转功能 | 支持Banner配置跳转 |
| 配置内容 | Banner图、图片文字、Banner顺序 |
| 自动切换 | 间隔2秒 |

#### Emoji Chicken入口

| 配置项 | 说明 |
|--------|------|
| 功能 | Emoji合成功能入口 |
| 展示内容 | 视频封面图、视频/图片内容 |
| 数据来源 | 本地存储，每次打开自动加载 |
| 风格名称 | 服务端下发 |

---

### 4.2 Emoji功能（P0）

#### 功能流程

```
上传图片 → 图片检测 → 选择风格 → 生成Loading → 完成页
```

#### 4.2.1 上传图片

| 项目 | 说明 |
|------|------|
| 触发方式 | 点击上传，调起本地相册 |
| 正面示例 | 正面完整头像 |
| 反面示例 | 遮挡人像、侧面人像、多人照片 |

#### 4.2.2 图片检测

| 项目 | 说明 |
|------|------|
| 检测时机 | 上传图片后自动检测 |
| 通过处理 | 自动进入【选择风格】 |
| 不通过处理 | 停留当前页面，提示图片有问题 |

#### 4.2.3 选择风格

- 风格列表由服务端下发
- 技术文档：[阿里云Emoji API](https://help.aliyun.com/zh/model-studio/emoji-api?spm=a2c4g.11186623.help-menu-2400256.d_2_3_13_1.6d6972dac0zxoX&scm=20140722.H_2865374._.OR_help-T_cn~zh-V_1#552aacbdadtaq)

#### 4.2.4 生成Loading

| 项目 | 说明 |
|------|------|
| 动画效果 | 加载动画，不展示真实生成过程 |
| 进度策略 | 前90%进度快，最后10%一直等待完成 |

#### 4.2.5 完成页

| 功能 | 说明 |
|------|------|
| 视频展示 | 加载当前生成视频，不展示视频时长 |
| 返回上一步 | 返回到选择风格页面 |
| 完成 | 点击完成，关闭页面，返回上级页面 |
| 重新生成 | 返回到Loading动画，扣减积分 |
| 分享 | V1.0可不开发 |
| 点赞 | 当前视频点赞，ICON变化为点赞态 |

#### 4.2.6 积分消耗规则

| 项目 | 说明 |
|------|------|
| 消耗时机 | 生成任务开启时 |
| 消耗数量 | 200积分（可由Server配置调整） |
| 数据存储 | 以设备ID存储积分数据（先本地-再Server） |

| 配置项 | 说明 |
|--------|------|

| 交互 | 说明 |

### 4.4 会员主页（P1）

#### 会员状态

| 状态 | 展示 |
|------|------|
| 是会员 | 特殊会员标识 |
| 非会员 | 标识置灰 |

#### 历史概览

| 项目 | 说明 |
|------|------|
| 内容 | 提交一次生成时的内容 |
| 分类 | 图片和视频 |
| 视频展示 | 第一帧作为封面图，有视频标识 |

#### 积分功能

| 交互 | 说明 |
|------|------|
| 点击积分 | 跳转积分详情页 |
| 选择积分包 | 积分包对应奖励积分由Server下发配置 |
| 点击购买 | 调起iOS/Android购买页 |

---

### 4.5 付费引导页

#### 页面布局

| 区域 | 说明 |
|------|------|
| 顶部效果图 | 支持自动轮播（视频或图片） |
| 视频播放 | 自动播放，播放完成后切换到下一个内容 |
| 图片停留 | 3秒后切换 |
| 文案 | 主标题 + 副标题 |

设计参考：[Gemini沉浸式效果](https://gemini.google.com/share/f46c49fecf10)

#### 订阅方案

| 项目 | 说明 |
|------|------|
| 默认选择 | 第一项 |
| 点击操作 | 调起iOS/Google订阅能力，完成订阅 |

#### 权益说明文案

- Enhance, Colorize, Animate faster
- Remove All Function Limits
- Extra Points Discount
- No ads

---

### 4.6 暂不实现功能

#### 热门作品（P1，V1.0不做）

- 热门作品由服务端下发视频
- 客户端请求，随机展示
- 热门内容库由运营提供，服务端可添加视频库内容
- 点击直接打开图片或视频（用系统打开能力）

#### 登录（P2，不做）

- 用户登录功能暂不实现
